# Smart-City-AI

A model built using detectron2 to identify the following Visual pollution types:

● GRAFFITI
● FADED SIGNAGE
● POTHOLES
● GARBAGE
● CONSTRUCTION ROAD
● BROKEN_SIGNAGE
● BAD STREETLIGHT
● BAD BILLBOARD
● SAND ON ROAD
● CLUTTER_SIDEWALK
● UNKEPT_FACADE
